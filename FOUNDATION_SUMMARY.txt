🔑 KEY FEATURES
===============

1. INTELLIGENT FALLBACK
   Open-Meteo → NOAA → METAR
   Automatically tries next source on failure. User gets data or appropriate error.

2. SMART CACHING  
   30-minute default TTL. Same coordinates = cached result (500x faster).
   Configurable: WeatherAggregator(cache_ttl_minutes=60)

3. ENSEMBLE CONFIDENCE
   Multi-member ensemble forecasts show prediction certainty.
   Temperature spread = forecast uncertainty
   confidence['temperature_confidence'] = 0.0-1.0

4. ML-READY FEATURES
   WeatherFeatureExtractor automatically generates features for your model:
   - Current conditions dict
   - Hourly/daily statistics
   - Ensemble spreads & confidence
   - Market-specific calculations (temp thresholds, wind events)

5. FORECAST VALIDATION
   Compare predictions to METAR observations (post-resolution):
   - MAE (Mean Absolute Error)
   - RMSE (Root Mean Square Error)
   - Bias (systematic over/under prediction)

📊 DATA MODELS (Normalized Across All Sources)
===============================================

LocationWeatherData
  ├─ current: CurrentWeather

⚙️ INTEGRATION WITH KALSHI BOT
==============================

Weather Foundation (this!)
         ↓
  Extract Features (WeatherFeatureExtractor)
         ↓
  Your ML/Prediction Model
         ↓
  Generate Predictions
         ↓
  Place Kalshi Bets
         ↓
  Validate Against Observations (post-resolution)

Example:
  from weather_aggregator import WeatherAggregator
  from weather_utils import WeatherFeatureExtractor
  
  agg = WeatherAggregator()
  weather = agg.get_complete_weather_data(market.lat, market.lon)
  features = WeatherFeatureExtractor.current_conditions_dict(weather)
  prediction = your_model.predict(features)
  kalshi.place_bet(market, prediction)

💡 COMMON USE CASES
===================

Current Weather:
  current = agg.get_current_weather(lat, lon)
  → CurrentWeather object

Forecast Next 7 Days (Hourly):
  hourly = agg.get_hourly_forecast(lat, lon, days=7)
  → List[ForecastPoint] (168 points)

Forecast Next 30 Days (Daily):
  daily = agg.get_daily_forecast(lat, lon, days=30)
  → List[ForecastPoint] (30 points)

Ensemble with Confidence:
  ensemble = agg.get_ensemble_forecast(lat, lon)
  confidence = agg.ensemble_confidence_score(ensemble)
  → Confidence 0-100%, mean spread

Everything at Once:
  weather = agg.get_complete_weather_data(lat, lon)
  → LocationWeatherData (all sources, current + forecast + historical)

📍 CUSTOMIZE FOR YOUR MARKETS
=============================

Define location configs:
  MARKETS = {
      "NYC": {"lat": 40.7128, "lon": -74.0060, "station": "KJFK"},
      "LA": {"lat": 34.0522, "lon": -118.2437, "station": "KLAX"},
      "Chicago": {"lat": 41.8781, "lon": -87.6298, "station": "KORD"},
  }

Then loop through:
  for market_name, coords in MARKETS.items():
      weather = agg.get_complete_weather_data(
          coords["lat"], coords["lon"],
          station_code=coords.get("station")
      )
